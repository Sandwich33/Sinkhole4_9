데미지 처리 이벤트.

어떠한 종류의 데미지라도 데미지를 입는다면 들어오는 이벤트다.

팀킬 가능 여부를 체크한 뒤 Demage를 Hp에 적용한다.

데미지 처리 후 죽을 경우 적절한 애니메이션을 실행한다.

주변 동료들에게 공격받은 사실을 알림.
