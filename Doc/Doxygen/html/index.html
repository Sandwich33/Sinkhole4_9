Sinkhole

Unreal Engine 4 로 제작한 게임으로, 주요 구조의 문서. ;

해당 문서는 'doxygen' 이라는 프로그램으로 제작되었으며, Unreal Engine 4 의 Blueprint 소스는 번역이 불가능하여 임시로 제작한 Cpp 코드로 문서를 제작 했습니다.

해당 문서 및 문서 제작용 프로젝트에는 Unreal Engine 4의 기본 제공 클래스 내용을 담고 있지 않습니다.

Unreal Engine 4의 Blueprint class에는 Construction Script(컨스트럭션 스크립트), Event Graph, Macros 가 존재합니다. 이를 이 문서에서는 각각의 함수로 표현합니다.

  • Construction Script [https://docs.unrealengine.com/latest/KOR/Engine/Blueprints/UserGuide/UserConstructionScript/index.html]
  • Event Graph [https://docs.unrealengine.com/latest/KOR/Engine/Blueprints/UserGuide/EventGraph/index.html]
  • Event [https://docs.unrealengine.com/latest/KOR/Engine/Blueprints/UserGuide/Events/index.html]
  • Macros [https://docs.unrealengine.com/latest/KOR/Engine/Blueprints/UserGuide/Macros/index.html]

또, Unreal Engine 4의 Blueprint class에서는 한 함수가 여러 값을 리턴 할 수 있습니다. 이를 해당 문서에서는 '*'(call by reference) 를 사용하여 표현 합니다.

해당 프로젝트는 다음과 같은 코딩컨벤션을 지키는 것을 권장 합니다.

  • Class Name 의 첫글자는 대문자 이며 '_'를 붙이고 그 뒤에 상속받은 부모 클래스 명을 붙인다.
