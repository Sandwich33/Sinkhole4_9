서비스를 가지는 분기 노드

Service_updateMyLocaion과 Service_ScanEnemyCharacter, Service_Destination 서비스를 가지고 있다.

Parameters
myCharacter는주변에 적이 발견된다면 on, 아니면 flase로 됨
myLocation은플레이어의 위치를 실시간으로 업데이트함
Destination은명령이동시에 해당 목적지 위치를 저장한다.
distanceCheck는플레이어와 AI의 거리가 멀면 true, 아니면 false로 세팅된다.
targetToAttack은총든 AI사거리에 따라 발견하는 범위안에 적이 있다면 해당 적의 Object를 세팅
attackRange는AI가 들고있는 무기를 검색해서, 무기에 따라 사거리 정해주는 초기값.
bullet은현재 장전되어 있는 탄 수를 업데이트함.
SelectAI는플레이어가 팀원AI 중 하나를 선택하면 선택된 팀원AI의 Object를 세팅함.
Command는명령을 내렸다면 True, 아니면 False를 세팅함.
OriginaRotation은공격 후 이상해진 Rotation을 복구하기 위해 임시로 만든 AI들의 처음 Rotation 초기값.
